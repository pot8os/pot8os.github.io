主にCircleCIなんですがyml文法の話なのでGitHub Actionsや他でも使えると思います。
- keyのためにreference作る必要などなかった
```yml
steps:
  - restore_cache:
      keys:
        # これで ↓
        - &node_cache node-cache-v1-{{ checksum "package.json" }}
  - run: npm install
  - save_cache:
      key: *node_cache # ここで再利用できる
      path: node_module
```
- pathsはまとめて指定してもよい
```yml
  - save_cache:
      key: *foobar_cache
      paths: ["~/foo", "~/bar", "~/foobar"] # リストで書く必要ない 
```
